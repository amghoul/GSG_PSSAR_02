# Academic Data Integration and Analysis Pipeline

A Python data analysis pipeline utilizing `pandas` to load, inspect, clean, merge, and query multi-format research datasets (CSV, JSON, and Excel). The script processes information across researchers, academic publications, and project funding to uncover key institutional insights.

## Project Structure

Expected repository file layout:
```text
├── data/
│   ├── researchers.csv      # Demographics and status metadata
│   ├── publications.json    # Academic papers and citation records
│   └── funding.xlsx         # Financial grant and funding records
├── output/
│   └── clean_research_data.csv # Final curated dataset generated by the pipeline
├── main.py                  # Primary data analysis script
└── README.md                # Documentation

```

## Features

* **Multi-Format Data Ingestion:** Dynamic data loading supporting JSON, XLSX, and CSV formats.
* **Dynamic Diagnostics:** Reflective inspection of DataFrames using metadata attributes (`dtypes`, `describe`, `shape`, etc.).
* **Data Cleansing & Validation:** Filters missing attributes and invalid zero/negative numeric records.
* **Relational Merges:** Evaluates data continuity using both strict `inner` and permissive `left` relational joins.
* **Aggregated Analytics:** Performs targeted analytical grouping to isolate top performance metrics (citations, funding, and career longevity).

## Functional Overview

### 1. Data Processing Core

* `load_file(file_name)`: Parses files dynamically according to extension type and logs the initial records (`df.head()`).
* `get_file_stats(df)`: Loops through essential analytical criteria (`isnull`, `duplicated`, etc.) using dynamic attribute fetching via `getattr()`.
* `clean_funding(df)`: Drops records missing critical values, converts metrics to numeric types, and discards non-positive entries.
* `merge_func(df1, df2, df3, type)`: Chains multiple DataFrame joins over the unified foreign key `researcher_id`.

### 2. Analytical Findings

The pipeline evaluates and answers three core institutional questions:

1. **Highest Total Citations:** Identifies the researcher with the highest sum of academic citations.
2. **Top Funded Domain:** Tracks which research domain or `field` received the highest financial backing in CAD.
3. **Senior Active Staff:** Locates active researchers with the earliest registration year (`joined_year`).

## Output Generation

Upon successful completion, the script exports a consolidated, filtered table tracking all concurrent metrics into:
`output/clean_research_data.csv`
